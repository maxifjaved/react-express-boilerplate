React Starter Kit
=====================

### Usage

```
git clone https://github.com/maxifjaved/react-webpack-express-boilerplate.git
cd react-webpack-express-boilerplate/
npm install
npm run server
open http://localhost:3000
```

### Linting

This project includes React-friendly ESLint configuration.

```
npm run lint
```

### Customization
